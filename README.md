# Autonomous-County-Budget-Intelligence-Agent
An agentic AI system that analyzes New York State county government budget data using RAG, autonomous Python execution, and multi-agent orchestration. Built for AIM 490W Capstone, Farmingdale State College.
The system runs a 3-step agentic pipeline. In Step 1, the user’s question is sent to Claude along with RAG context retrieved from a ChromaDB vector store (1,575 embedded chunks covering budget category definitions, column descriptions, and category-to-query mappings). Claude then writes Python pandas code tailored to the exact dataset—it knows the column names, available counties, year range, and category structure because we pass all of that as metadata in the prompt.
In Step 2, the generated code is executed in a sandboxed Python environment with access to pandas, matplotlib, and the loaded DataFrame. The agent captures stdout, handles errors gracefully, and if a chart would help visualize the answer, it generates one and saves it to disk. In Step 3, the execution results—raw numbers, tables, and any generated charts—are sent back to Claude, which writes a 2–3 paragraph executive briefing suitable for a county analyst or decision-maker.
The end-to-end flow: user question → RAG retrieval from ChromaDB → Claude generates Python code → code executes against live data → Claude synthesizes results into a narrative briefing → output displays with charts. The user asks one question; the system completes five autonomous steps with no additional human input.
